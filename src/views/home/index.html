Authentication

The Campaign Monitor API requires authentication using a few different methods. createsend-node supports only one for now, the API key. 

var createsend = require('createsend-node');
var auth = { apiKey: 'your api key' };
var api = new createsend(auth);

Account

Campaign Monitor provides many account based operations. The sections list here follow the same order as those listed on the Campaign Monitor API page. For more details, check out https://www.campaignmonitor.com/api/account/

Getting your clients

Using this wrapper to get the list of clients will actually wrap each client in a Client object. This allows you to not only pull the clientId and name, it also provides all the common functions that may be performed on a client. 

api.account.getClients(function (err, clientList) {
    console.log('You have ' + clientList.length + ' clients.');
    for (var i = 0; i < clientList.length; i++) {
        console.log('    ' + clientList[i].name);
    }
})
